# IoTWild

Esse é o Repositório do Projeto IoTWild, contendo os códigos desenvolvidos pelo aluno de IC Eduardo Alves da Silva

Envio de Imagem -> Apresenta o código usado para um módulo ESP8266 em um computador enviar para outro um arquivo qualquer. (OBS : Os modulos precisam estar previamente conectados, e é necessário definir no codigo qual porta será usada para a comunicação com o módulo. Proximas versões não devem contar com isso)

Testes em Geral -> Alguns programas usados para testes de Roundtrip time/Error rate/Tempo de processamento, etc (OBS : Os modulos precisam estar previamente conectados, e é necessário definir no codigo qual porta será usada para a comunicação com o módulo. Proximas versões não devem contar com isso)
